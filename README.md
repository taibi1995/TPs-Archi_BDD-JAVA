# TPs Architecture BDD — Java

Educational Java projects illustrating database internals: block management, join algorithms and tests.

Build & Run
```bash
# Using Maven
mvn clean package
java -jar target/tp-archi-bdd-java-1.0.0.jar
```

Development
- Java 11+
- JUnit for tests
